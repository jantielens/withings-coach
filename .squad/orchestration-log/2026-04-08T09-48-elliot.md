### Evaluation

Researched 5 chat UI options + split-pane solutions:

**Chat UI Libraries:**
1. **Vercel AI SDK + shadcn/ui** ✅ Recommended
   - Native streaming support via `useChat` hook
   - React 19 native, Tailwind-first (shadcn/ui)
   - Bundle: ~25 kB gzipped
   - Active maintenance (Vercel)

2. **@chatscope/chat-ui-kit-react** — Rejected
   - No streaming support (blocker)
   - React 19 untested
   - Tailwind integration awkward
   - Bundle: 40–60 kB

3. **react-chat-elements** — Rejected
   - Too minimal, no streaming
   - Small community, uncertain React 19 support

4. **Custom Build** — Rejected
   - 2–3 weeks dev time
   - High maintenance burden
   - Redundant given Vercel AI SDK exists

5. **Stream Chat React (GetStream.io)** — Rejected
   - Overkill for single-user AI chat
   - Designed for team messaging, not AI coaching
   - Heavy bundle (100+ kB)

**Split-Pane Layout:**
- **Recommended:** `react-resizable-panels`
- ~8 kB gzipped, simple API, Tailwind-friendly
- Battle-tested, accessible, optional persistence
