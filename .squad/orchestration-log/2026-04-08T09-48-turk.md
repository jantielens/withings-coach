### Recommended: Vercel AI SDK

**Advantages:**
- Zero-boilerplate streaming (SSE auto-conversion)
- Managed identity first-class support
- React hook integration (`useChat`)
- Battle-tested in production
- TypeScript-first, good error handling

**Core packages:**
```
npm install ai @ai-sdk/azure @azure/identity
```

### Authentication: DefaultAzureCredential

**How it works:**
Tries credentials in order:
1. Environment variables (CI/CD)
2. Azure CLI (`az login` вЂ” local dev)
3. VSCode Azure extension (optional)
4. Managed identity (production on App Service)

**Local Development:**
```bash
az login  # Stores credentials locally
npm run dev  # DefaultAzureCredential auto-detects
```

**Production (Azure Container Apps / App Service):**
- Enable system-assigned managed identity on resource
- Assign `Cognitive Services User` RBAC role
- `DefaultAzureCredential` auto-detects environment
- No secrets stored, no key rotation needed

### Streaming Implementation

**API Route (`app/api/chat/route.ts`):**
- Fetch BP data, diary, context notes (DB queries)
- Build dynamic user context (markdown table)
- Inject into messages: [system prompt, user context, ...history]
- Call `streamText()` with model from `@ai-sdk/azure`
- Return `result.toDataStreamResponse()` (handles SSE + headers)

**Frontend (`components/chatbot.tsx`):**
- Use `useChat` hook from `ai/react`
- POST to `/api/chat` with full history
- Hook handles SSE parsing, message state, loading UI
